# Vue3小米手机后台管理系统 java110

## 项目概述

本项目是基于Vue3的纯前端小米手机后台管理系统。系统采用现代化的UI设计，功能完整，提供了一套无需后端数据库的解决方案，适用于各类前端项目模板或作为Vue3技术栈的进阶学习资源。

## 技术栈

- **前端框架**: Vue3
- **UI组件库**: Element Plus
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **HTTP请求**: Axios

## 功能模块

系统包含以下页面：

1. 登录页
2. 个人资料管理页
3. 手机分类管理页
4. 手机产品管理页
5. 手机详情查看弹窗
6. 404错误页面

核心功能包括：

- 管理员登录认证与Token验证
- 个人资料管理与头像上传
- 手机分类的完整CRUD（创建、读取、更新、删除）操作
- 手机产品的增删改查、搜索筛选、分页展示
- 手机详情弹窗查看与编辑

## 系统角色

- 管理员：负责登录系统，进行资料管理及手机产品管理

## 运行环境

- 现代浏览器

## 部署步骤

1. 安装Node.js环境
2. 使用npm或yarn安装项目依赖
3. 运行构建命令，构建生产环境的资源文件
4. 部署至合适的前端服务器

## 目录结构

```
.
├── public                     # 静态资源目录
├── src                        # 源代码目录
│   ├── assets                 # 资源文件，如样式、图片等
│   ├── components             # Vue组件
│   ├── views                  # 页面级Vue组件
│   ├── router                 # 路由配置
│   ├── store                  # 状态管理
│   ├── utils                  # 工具方法
│   ├── App.vue                # 主组件
│   └── main.js                # 入口文件
├── .gitignore                 # Git忽略文件配置
├── package.json               # 项目描述文件
└── vite.config.js             # Vite配置文件
```

## 核心亮点

- **响应式布局**：完美适配移动端设备
- **组件化开发**：支持父子组件通信，易于自定义组件开发
- **现代化界面**：渐变色UI设计，毛玻璃效果，动画交互丰富
- **权限控制**：路由守卫与状态管理确保访问安全
- **遵循Vue3最佳实践**：使用Composition API和setup语法，计算属性与侦听器等

## 状态管理与路由守卫

- **状态管理**：使用Pinia实现响应式的状态管理
- **路由守卫**：通过Vue Router守卫控制页面访问权限

本项目旨在为Vue3初学者提供一个进阶学习的平台，同时可以作为企业级后台管理系统的模板。
